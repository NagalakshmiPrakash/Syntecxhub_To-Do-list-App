# 📝 TO-DO LIST APP

## 📖 Overview

The TO-DO LIST APP is a modern and responsive task management web application developed using HTML, CSS, and JavaScript. The application allows users to efficiently organize daily activities through a clean, interactive, and user-friendly interface.

The project features a professional glassmorphism-inspired design, smooth animations, responsive layouts, and interactive task management functionalities that enhance the overall user experience across desktop and mobile devices.

---

## 🎯 Objectives

* Develop a modern and responsive TO-DO List application
* Practice frontend development using HTML, CSS, and JavaScript
* Implement DOM manipulation and event handling
* Store and manage task data using localStorage
* Create an interactive and user-friendly interface
* Improve practical web development and UI design skills

---

## ✨ Features

* Add new tasks instantly
* Mark tasks as completed
* Delete tasks easily
* Persistent task storage using localStorage
* Responsive design for all devices
* Smooth animations and modern UI
* Real-time task statistics
* Keyboard support using the Enter key

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

---

## 💻 Project Functionality

Users can add, complete, and remove tasks dynamically. Completed tasks are visually distinguished with checkmark indicators and strikethrough styling. The application stores task data using localStorage, ensuring tasks remain saved even after refreshing or reopening the browser.
